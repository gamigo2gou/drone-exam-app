   - 「CSVエクスポート」で `review_export.csv` をダウンロード
   - `answer` が変更された問題を各 JSON ファイルに手動または
     スクリプトで反映する
   - 反映後に `scripts/shuffle-answers.mjs` は**再実行しない**
     （正解位置が再度ランダム化されてしまう）

3. **GitHub に push**
   - JSON 更新後に `git add / commit / push`
   - Vercel が自動で再デプロイ

## 技術スタック

| 項目 | 内容 |
|------|------|
| フレームワーク | React 18 + Vite + TypeScript |
| スタイル | Tailwind CSS v4（`@tailwindcss/vite`） |
| ルーティング | React Router v6 |
| 状態管理 | Zustand + persist（localStorage） |
| PWA | vite-plugin-pwa + Workbox |

### localStorage キー一覧

| キー | 用途 |
|------|------|
| `drone-quiz-store` | 章別演習の進捗（正答率） |
| `drone-exam-history` | 模擬試験の受験履歴（最大10件） |
| `drone-progress-store` | 問題ごとの正誤統計・学習日記録 |
| `drone-admin-reviews` | 監修差分（answer変更・status） |

### 問題データ

```
public/questions/
  index.json              # 章一覧（id: ch1〜ch4）
  chapter1_rules.json     # 第1章 規則（75問）
  chapter2_system.json    # 第2章 システム（135問）
  chapter3_pilot.json     # 第3章 操縦者（30問）
  chapter4_risk.json      # 第4章 リスク管理（60問）
```

- `choices` は3択（0-indexed配列）
- `answer` は1-based（1/2/3）
- chapter2〜4は `scripts/shuffle-answers.mjs` で選択肢をシャッフル済み

### 模擬試験の出題配分

| 章 | 問題数 |
|----|--------|
| 第1章 | 13問（約25%） |
| 第2章 | 22問（約45%） |
| 第3章 | 5問（約10%） |
| 第4章 | 10問（約20%） |
| 合計 | **50問** |

合格基準：40問以上正解（80%）、制限時間：30分
